#  Dragon Warrior

A 2D action-adventure game developed in Unity using C#. The player takes control of a brave warrior on a quest through dangerous environments, battling enemies, avoiding hazards and collecting items to complete the adventure.

---

##  Features

- Smooth 2D player movement
- Enemy combat
- Health system
- Health collection
- Animated characters
- Sound effects
- Camera movement
- Multiple levels
- Game Over and Victory screens

---

##  Built With

- Unity
- C#
- Visual Studio
